# darksun

d20 System amalgamation of [athas.org](athas.org) books with [System Reference Document](https://www.d20srd.org/index.htm) already available under Open Game License 1.0a.

## athas.org Books used

- [Dark Sun 3](http://athas.org/products/ds3)
- [Faces of the Forgotten North](http://athas.org/products/ffn)
- [City State of Draj](http://athas.org/products/csod)
- [Life-Shaping Handbook](http://athas.org/products/lsh)
- [Prestige Class Appendix, Volume I](http://athas.org/products/prc1)
- [Prestige Class Appendix, Volume II](http://athas.org/products/prc2)
- [Terrors of the Dead Lands](http://athas.org/products/totdl)
- [Trade Lords](http://athas.org/products/tradelords)
